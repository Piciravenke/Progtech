# Progtech

Programozási technológiákra és környezetekre írt JavaFx-es alkalmazás

Az alkalmazásról röviden: Egy számológép, ami képes az alapmûveletek elvégzésére, mûveleti sorrend betartása mellett, illetve 2-es,8-as és 16-os számrendszerekbe való átváltásra. Az elvégzett számolásokat meg lehet tekinteni a kapcsolódó adatbázisban, a History gomb segítségével.

Rendszerkövetelmény:
- Java 8 JDK a számítógépen.
- Apache Maven
- Az ojdbc7 és ucp com.oracle.jdbc termékek a lokális maven tárolóban.
